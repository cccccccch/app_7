# 风机盘管控制器 & 空调控制器

这两个设备都是用的同一个page(即同一套UI，用`zigbeeType`区分)，目录 `aircondition`

# 区别：

## 风机盘管控制器
1. 调节范围在16-30度
2. 不具备儿童模式
3. 不具备除湿功能

wiki: http://wiki.wingto.com/pages/viewpage.action?pageId=24348634

## 空调控制器

1. 调节范围在16-32度
2. 具备儿童模式
3. 具备除湿功能

wiki: http://wiki.wingto.com/pages/viewpage.action?pageId=24348882
